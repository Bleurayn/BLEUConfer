# BLEUConfer – Sovereign Video Conferencing OS

**Live MVP**: https://bleurayn.github.io/bleuconfer-sovereign-platform/  
**Permanent DOI**: Coming Q4 2025

End-to-end encrypted, deepfake-resistant video conferencing with live smart contracts and automatic archival.

### Features (MVP)
- WebRTC peer-to-peer HD video & audio
- Sovereign Peer ID system (foundation for AntiSIM)
- Zero servers — fully decentralized
- Ready for BLEUVault, AvaSign, BLEUCalendar integration

This is a public demonstration only.  
The full commercial platform — including ML-powered AntiSIM, BLEUVault archival, live contracts, and heritage remembrance — is 100% closed-source and in active development.
